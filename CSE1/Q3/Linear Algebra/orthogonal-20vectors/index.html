Definition

Let ﻿W⊂RnW\subset\mathbb{R}^nW⊂Rn﻿

  • A vector ﻿xxx﻿ in ﻿Rn\mathbb{R}^nRn﻿ is orthogonal to ﻿WWW﻿ if ﻿x⊥wx\perp wx⊥w﻿ for each ﻿www﻿ in ﻿WWW﻿ (notated by ﻿x⊥Wx\perp Wx⊥W﻿)
  • The orthogonal complement ﻿W⊥W^{\perp}W⊥﻿ of ﻿WWW﻿ is the set of all vectors ﻿xxx﻿ that are orthogonal to ﻿WWW﻿: ﻿{W⊥={x∈Rn∣x⊥w ∀w∈W}\left\{W^{\perp}=\left\{x\in\mathbb{R}^n\right|x\perp w\ \forall w\in W\right\}{W⊥={x∈Rn∣x⊥w ∀w∈W}﻿

Let ﻿W⊂RnW\subset\mathbb{R}^nW⊂Rn﻿ be a subspace

Orthonormal sets and bases

  • An orthonormal set is an orthogonal set ﻿{u1, …, up}\left\{u_1,\ \dots,\ u_p\right\}{u1​, …, up​}﻿ of unit vectors: ﻿ui⋅uj=0u_i\cdot u_j=0ui​⋅uj​=0﻿ if ﻿i≠ji\ne ji​=j﻿, ﻿111﻿ if ﻿i=ji=ji=j﻿
  • An orthonormal basis for a subspace ﻿WWW﻿ is a basis that is also an orthonormal set
  • An orthogonal matrix is a square matrix ﻿UUU﻿ for which ﻿UTU=IU^TU=IUTU=I﻿ (so for which the columns are orthonormal)


So, the following statements are equivalent for a square matrix ﻿UUU﻿:

  • ﻿UUU﻿ is an orthogonal matrix
  • ﻿UTU=IU^TU=IUTU=I﻿
  • ﻿UUT=IUU^T=IUUT=I﻿
  • ﻿U−1=UTU^{-1}=U^TU−1=UT﻿
  • The columns of ﻿UUU﻿ are orthonormal
  • The rows of ﻿UUU﻿ are orthonormal

Let ﻿UUU﻿ be an ﻿m×nm\times nm×n﻿ transformation matrix with orthonormal columns. Then the transformation ﻿T(x)=UxT\left(x\right)=UxT(x)=Ux﻿ preserves the inner product: ﻿T(x)⋅T(y)=x⋅y ∀x, y∈RnT\left(x\right)\cdot T\left(y\right)=x\cdot y\ \forall x,\ y\in\mathbb{R}^nT(x)⋅T(y)=x⋅y ∀x, y∈Rn﻿.

QR factorization

Let ﻿AAA﻿ be an ﻿m×nm\times nm×n﻿ matrix with linearly independent columns. Then there exists an ﻿m×nm\times nm×n﻿ matrix ﻿QQQ﻿ with orthonormal columns and an ﻿n×nn\times nn×n﻿ upper triangular matrix ﻿RRR﻿ with positive diagonal entries such that ﻿A=QRA=QRA=QR﻿ (which implies that ﻿R=QTQR=QTAR=Q^TQR=Q^TAR=QTQR=QTA﻿). Thus ﻿QQQ﻿ can be found using the Gram-Schmidt process and ﻿RRR﻿ can be found by multiplying matrices.


QR algorithm

Let ﻿AAA﻿ be a square matrix with only real eigenvalues

  1. Let ﻿A0=AA_0=AA0​=A﻿

  4. Let ﻿Ak+1=RkQkA_{k+1}=R_kQ_kAk+1​=Rk​Qk​﻿

  • Let AAA be an m×nm\times nm×n matrix and b∈Rmb\in \R^mb∈Rm. Then the least-squares solution of Ax=bAx=bAx=b is an x^∈Rn\hat{x}\in \R^nx^∈Rn such that ∣∣b−Ax^∣∣≤∣∣b−Ax∣∣||b-A\hat{x}||\leq ||b-Ax||∣∣b−Ax^∣∣≤∣∣b−Ax∣∣ for all x∈Rnx\in\R^nx∈Rn.
  • The distance ∣∣Ax^−b∣∣||A\hat{x}-b||∣∣Ax^−b∣∣ is called the least-squares error
  • The set of least-squares solutions of Ax=bAx=bAx=b coincides with the nonempty set of solutions of the normal equations ATAx=ATbA^TAx=A^TbATAx=ATb.
