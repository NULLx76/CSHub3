Determinants

The determinant of an ﻿n×nn\times nn×n﻿ matrix ﻿AAA﻿, with ﻿n≥2n\ge2n≥2﻿, can be computed by a cofactor expansion across any row or column.


The cofactor expansion along row ﻿iii﻿ is given by:

﻿det⁡A=ai1Ci2+ai2Ci2+⋯+ainCin\det A=a_{i1}C_{i2}+a_{i2}C_{i2}+\dots+a_{in}C_{in}detA=ai1​Ci2​+ai2​Ci2​+⋯+ain​Cin​﻿ (where ﻿aina_{in}ain​﻿ is the ﻿iii﻿th element in the ﻿nnn﻿th column of matrix ﻿AAA﻿)


The cofactor expansion along column ﻿jjj﻿﻿iii﻿ is given by:

﻿det⁡A=a1jC1j+a2jC2j+⋯+anjCnj\det A=a_{1j}C_{1j}+a_{2j}C_{2j}+\dots+a_{nj}C_{nj}detA=a1j​C1j​+a2j​C2j​+⋯+anj​Cnj​﻿


Properties

  • ﻿det⁡(AB)=det⁡(BA)=det⁡(A)det⁡(B)\det\left(AB\right)=\det\left(BA\right)=\det\left(A\right)\det\left(B\right)det(AB)=det(BA)=det(A)det(B)﻿
  • ﻿det⁡(A)=det⁡(−A)\det\left(A\right)=\det\left(-A\right)det(A)=det(−A)﻿ when the matrix size is even
  • ﻿det⁡(An)=det⁡(A)n\det\left(A^n\right)=\det\left(A\right)^ndet(An)=det(A)n﻿
  • If ﻿T(x)=AxT\left(x\right)=AxT(x)=Ax﻿ is a linear transformation, the area of the image of a parallelogram ﻿PPP﻿ equals: ﻿Area(T(P))=∣det⁡(A)∣Area(P)Area\left(T\left(P\right)\right)=\left|\det\left(A\right)\right|Area\left(P\right)Area(T(P))=∣det(A)∣Area(P)﻿
  • If ﻿AAA﻿ and ﻿BBB﻿ are ﻿n×nn\times nn×n﻿ matrices, ﻿det⁡(AB)=(det⁡A)(det⁡B)\det\left(AB\right)=\left(\det A\right)\left(\det B\right)det(AB)=(detA)(detB)﻿.
  • If ﻿AAA﻿ is invertible, then ﻿det⁡(A−1)=1det⁡A\det\left(A^{-1}\right)=\frac{1}{\det A}det(A−1)=detA1​﻿
  • Subtracting and adding a multiple of rows in a matrix does not change the value of the determinant
  • When all entries in any row or column of the matrix ﻿AAA﻿ are multiplied with scalar ﻿kkk﻿, the determinant will be ﻿k⋅det⁡(A)k\cdot\det\left(A\right)k⋅det(A)﻿

Triangular matrices

The determinant of an ﻿n×nn\times nn×n﻿ triangular matrix ﻿AAA﻿ is equal to ﻿det⁡A=a11a22⋯ann\det A=a_{11}a_{22}\cdots a_{nn}detA=a11​a22​⋯ann​﻿


Row/column operations and determinants

Let ﻿AAA﻿ be a ﻿n×nn\times nn×n﻿ matrix. Then:

  1. If a multiple of one row is added to another row to produce matrix ﻿BBB﻿, then ﻿det⁡B=det⁡A\det B=\det AdetB=detA﻿
  2. If two rows of ﻿AAA﻿ are interchanged to produce ﻿BBB﻿, then ﻿det⁡B=−det⁡A\det B=-\det AdetB=−detA﻿
  3. If one row of ﻿AAA﻿ is scaled with a factor ﻿kkk﻿ to produce ﻿BBB﻿, then ﻿det⁡B=k⋅det⁡A\det B=k\cdot\det AdetB=k⋅detA﻿


The same holds for column operations.


Efficiently computing the determinant


  • Create zeros in a column by row operations (or zeros in a row by column operations)
  • Expand across the column (or row) with the zeros

The formula for that would be: ﻿xi=det⁡(Ai)det⁡(A)x_i=\frac{\det\left(A_i\right)}{\det\left(A\right)}xi​=det(A)det(Ai​)​﻿, where ﻿AiA_iAi​﻿ is the matrix ﻿AAA﻿ with the ﻿iii﻿th column replaced by the solution (﻿bbb﻿). If ﻿AAA﻿ is a ﻿2×22\times22×2﻿ matrix, ﻿∣det⁡(A)∣\left|\det\left(A\right)\right|∣det(A)∣﻿ is equal to the area of the parallelogram determined by the columns of ﻿AAA﻿


If ﻿AAA﻿ is a ﻿3×33\times33×3﻿ matrix, ﻿∣det⁡(A)∣\left|\det\left(A\right)\right|∣det(A)∣﻿ is equal to the volume of the parallelepiped determined by the columns of ﻿AAA﻿
