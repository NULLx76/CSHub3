Matrix transformations

A matrix ﻿AAA﻿ of size ﻿m×nm\times nm×n﻿ defines a transformation ﻿TTT﻿ defined by ﻿T: Rn→RmT:\ \mathbb{R}^n\to\mathbb{R}^mT: Rn→Rm﻿, ﻿T(x)=AxT\left(x\right)=AxT(x)=Ax﻿. A transformation is a rule that assigns to each vector ﻿xxx﻿ in ﻿Rn\mathbb{R}^nRn﻿, a vector ﻿T(x)T\left(x\right)T(x)﻿ in ﻿Rm\mathbb{R}^mRm﻿.


Here ﻿Rn\mathbb{R}^nRn﻿ is called the domain of ﻿TTT﻿, ﻿Rm\mathbb{R}^mRm﻿ is called the codomain of ﻿TTT﻿.


For an ﻿xxx﻿ in ﻿Rn\mathbb{R}^nRn﻿, the vector ﻿T(x)T\left(x\right)T(x)﻿ in ﻿Rm\mathbb{R}^mRm﻿ is called the image of ﻿xxx﻿, the set of all images is called the range

  • ﻿T(x+y)=T(x)+T(y)T\left(x+y\right)=T\left(x\right)+T\left(y\right)T(x+y)=T(x)+T(y)﻿
  • ﻿T(cx)=cT(x)T\left(cx\right)=cT\left(x\right)T(cx)=cT(x)﻿


Linear transformations

A transformation ﻿T: Rn→RmT:\ \mathbb{R}^n\to\mathbb{R}^mT: Rn→Rm﻿ is called a linear transformation if:

  • ﻿T(u+v)=T(u)+T(v)T\left(u+v\right)=T\left(u\right)+T\left(v\right)T(u+v)=T(u)+T(v)﻿ for all ﻿u, vu,\ vu, v﻿ in the domain of ﻿TTT﻿
  • ﻿T(cu)=cT(u)T\left(cu\right)=cT\left(u\right)T(cu)=cT(u)﻿ for all scalars ﻿ccc﻿ and ﻿uuu﻿ in the domain of ﻿TTT﻿


For any ﻿m ×nm\ \times nm ×n﻿ matrix ﻿AAA﻿ , the matrix transformation ﻿T(x)=AxT\left(x\right)=AxT(x)=Ax﻿ is a linear transformation ﻿T: Rn→RmT:\ \mathbb{R}^n\to\mathbb{R}^mT: Rn→Rm﻿. The result of a transformation from ﻿Rn\mathbb{R}^nRn﻿ to ﻿Rm\mathbb{R}^mRm﻿ is a ﻿m×nm\times nm×n﻿ matrix (﻿mmm﻿ rows with ﻿nnn﻿ columns)


It is easiest to think of a transformation as what happens with the following points in ﻿R2\mathbb{R}^2R2﻿: ﻿[1001]\begin{bmatrix}1&amp;0\\0&amp;1\end{bmatrix}[10​01​]﻿.

Properties of linear transformations

  • The image of a straight line is a straight line or a point
  • Parallel lines get mapped to parallel lines (or points)

One-to-one

﻿TTT﻿ is one-to-one if and only if the following statements hold;

  • ﻿T(0)=0T\left(0\right)=0T(0)=0﻿
  • The columns of ﻿AAA﻿ are independent
  • Every column of ﻿AAA﻿ contains a pivot


Onto

﻿TTT﻿ is onto if and only if the following statements hold:

  • The columns of ﻿AAA﻿ span ﻿Rm\mathbb{R}^mRm﻿
  • Every row of ﻿AAA﻿ contains a pivot position
