Function definition

A function from ﻿AAA﻿ to ﻿BBB﻿, detonated as ﻿f : A→Bf\ :\ A\to Bf : A→B﻿, is a relation from ﻿AAA﻿ (domain) to ﻿BBB﻿ (co-domain / range), that satisfies two conditions:

  • Every element of ﻿AAA﻿ is related to some element of ﻿BBB﻿
  • No element of ﻿AAA﻿ is related to more than one element of ﻿BBB﻿

A function is not well defined if it violates one of these conditions.


A function is one-to-one iff ﻿f(a)=f(b)⇔ a=bf\left(a\right)=f\left(b\right)\Leftrightarrow\ a=bf(a)=f(b)⇔ a=b﻿, to put it more informally, a function is one-to-one if and only if no horizontal line intersects its graph more than once.


Inverse function

Suppose ﻿fff﻿ is a function with domain ﻿DDD﻿ and range ﻿RRR﻿. A function ﻿ggg﻿ is the inverse of ﻿fff﻿ (on ﻿RRR﻿) if for all ﻿xxx﻿ in ﻿DDD﻿ and ﻿yyy﻿ in ﻿RRR﻿ the following holds: ﻿f(x)=y⇔ x=g(y)f\left(x\right)=y\Leftrightarrow\ x=g\left(y\right)f(x)=y⇔ x=g(y)﻿.


The domain of an inverse function is the same as the range of the 'normal' function. Similarly, the range of the inverse function is the domain of the 'normal' function.


You can only find an inverse of a function if it is one-to-one.


The notation of an inverse function is ﻿f−1f^{-1}f−1﻿. Note that ﻿f(f−1(y))=yf\left(f^{-1}\left(y\right)\right)=yf(f−1(y))=y﻿ and ﻿f−1(f(x))= xf^{-1}\left(f\left(x\right)\right)=\ xf−1(f(x))= x﻿ (cancellation equations)


You can find the inverse the following way:

  • start with ﻿y=f(x)y=f\left(x\right)y=f(x)﻿
  • solve this equation for ﻿xxx﻿ in terms of ﻿yyy﻿
  • interchange the role of ﻿xxx﻿ and ﻿yyy﻿, we then have ﻿y=f−1(x)y=f^{-1}\left(x\right)y=f−1(x)﻿

Functions with multiple variables

Level curves

The level curves of a function ﻿fff﻿ of two variables are the curves in ﻿R2\mathbb{R}^2R2﻿ with equations ﻿f(x, y)=cf\left(x,\ y\right)=cf(x, y)=c﻿.

A plot with several level curves is called a contour map.
