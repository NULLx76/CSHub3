Discrete distributions

Let ﻿Ω\OmegaΩ﻿ be a sample space. A discrete random variable is a function ﻿X: Ω→R X:\ \Omega\to\mathbb{R}\ X: Ω→R ﻿ that takes on a finite number of values ﻿a1,a2,…,ana_1,a_2,\dots,a_na1​,a2​,…,an​﻿ or an infinite number of values ﻿a1,a2,…a_1,a_2,\dotsa1​,a2​,…﻿


A discrete random variable ﻿XXX﻿ "transforms" a sample space ﻿Ω\OmegaΩ﻿ to a more "tangible" sample space ﻿Ω~\tilde{\Omega}Ω~﻿, whose events are more directly related to what you are interested in.


Probability mass function

The probability mass function ﻿ppp﻿ of a discrete random variable ﻿XXX﻿ is the function ﻿p: R→[0, 1]p:\ \mathbb{R}\to\left[0,\ 1\right]p: R→[0, 1]﻿, defined by ﻿p(a)=P(X=a)p\left(a\right)=P\left(X=a\right)p(a)=P(X=a)﻿ for ﻿−∞&lt;a&lt;∞-\infty&lt;a&lt;\infty−∞<a<∞﻿


Or in different words: a probability mass function gives the probability that a discrete random value is exactly equal to some value.


Distribution function

The (cumulative) distribution function ﻿FFF﻿ of a random variable ﻿XXX﻿ is the function ﻿F: R→[0, 1]F:\ \mathbb{R}\to\left[0,\ 1\right]F: R→[0, 1]﻿, defined by ﻿F(a)=P(X≤a)F\left(a\right)=P\left(X\le a\right)F(a)=P(X≤a)﻿ for ﻿−∞&lt;a&lt;∞-\infty&lt;a&lt;\infty−∞<a<∞﻿ . Or in different words: a distribution function gives the probability that a discrete random value is exactly equal to or smaller than some value.

Poisson distribution

Informal: how many times will something happen in a time interval?

When to use it?: if a mean/average is given and you want to know the probability of ﻿nnn﻿ events happening in a given time.


Things that are described well by a Poisson distribution:

  • ﻿kkk﻿ phone calls arriving at a call center per hour (random)
  • ﻿kkk﻿ floods per 100 years


Things that are NOT described well by a Poisson distribution:

  • ﻿kkk﻿ banks going bankrupt in a year (clustered)
  • ﻿kkk﻿ buses arriving per hour (regular)


The Poisson process is good at modelling the number of times an independent event occurs in an interval of time or space where the average frequency is known. It is useful to ask how many times an event happened, but not how many events didn't happen.


Calls arrive at random times ﻿X1,X2,…X_1,X_2,\dotsX1​,X2​,…﻿ at the call center in a time interval ﻿[0, t]\left[0,\ t\right][0, t]﻿. The two assumptions we make of these calls are:

  • The rate ﻿λ\lambdaλ﻿ at which arrivals occur is constant over time: in a subinterval of length ﻿uuu﻿ the expectation of the number of phone calls is ﻿λu\lambda uλu﻿ (homogeneity)
  • The numbers of arrivals in disjoint time intervals are independent random variables (independence)


A discrete random variable ﻿XXX﻿ has a Poisson distribution with parameter ﻿μ\muμ﻿ (average number of events per interval), where ﻿μ&gt;0\mu&gt;0μ>0﻿ if its probability mass function ﻿ppp﻿ is given by:

﻿p(k)=P(X=k)=μkk!e−μp\left(k\right)=P\left(X=k\right)=\frac{\mu^k}{k!}e^{-\mu}p(k)=P(X=k)=k!μk​e−μ﻿ for ﻿k=0,1,2,…k=0,1,2,\dotsk=0,1,2,…﻿ (the actual number of successes that occur in a specified region). Here, ﻿μ=λt\mu=\lambda tμ=λt﻿


This is denoted as ﻿X∼Pois(μ)X\sim Pois\left(\mu\right)X∼Pois(μ)﻿
