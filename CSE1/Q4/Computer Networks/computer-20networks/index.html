Assessment

The course grade is the grade of the final exam. The exam will contain roughly 25 questions, several multiple choice questions and usually 12 questions that require calculations (expect calculations like CRC checksums, Hamming codes, routing tables, and several bits of RSA encryption)


To be eligible for the exam, you need to pass the lab:

  • You have to pass assignment 1, 4 and at least one of 2, 3, 5
  • Passing an assignment means all quizzes and programming exercises need to be completed on Weblab in the corresponding assignment directory


Probably on the exam

  • CRC
  • Hamming code calculations
  • Binary division without carry
  • RSA calculations
  • Shortest path on a routing graph (Dijkstra's Algorithm)
  • How many errors can you detect / correct with hamming codes?
  • Network bits / host bits IPv4
  • DHT Lookup
