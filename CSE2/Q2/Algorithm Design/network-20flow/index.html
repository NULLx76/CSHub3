Max-flow algorithm

To solve the max-flow problem, we can find the maximum value by:

  1. Start with ﻿∀e∈E: f(e)=0\forall e\in E:\ f\left(e\right)=0∀e∈E: f(e)=0﻿
  2. Find an ﻿s−ts-ts−t﻿ path where each edge has ﻿f(e)&lt;c(e)f\left(e\right)&lt;c\left(e\right)f(e)<c(e)﻿
  3. Augment flow along path ﻿PPP﻿
  4. Go back to step ﻿222﻿ while such a path exists
  5. Also allow decreasing the flow on an edge


We do (5) by changing our graph to a residual graph, with residual edges. For every edge in the graph, it will create a residual edge with a capacity of the current flow. More formal:

  2. Start with ﻿Δ\DeltaΔ﻿ as the smallest power of ﻿222﻿ that is larger than the largest capacity leaving ﻿sss﻿, called ﻿c∗c^{\ast}c∗﻿. We do that so we can divide by 2 and keep ending up with an integer
  3. Ignore all edges with ﻿c(e)&lt;Δc\left(e\right)&lt;\Deltac(e)<Δ﻿

  6. Go back to step 4 while such a path exists
  7. Half ﻿Δ\DeltaΔ﻿ while it is larger than 1 and go back to step 2
