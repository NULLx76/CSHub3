AI

from a philosophical perspective

  • Weak AI hypothesis: Machines may act intelligently, without:
  • actually thinking
  • having a mind, i.e., mental states of self-awareness


  • Strong AI hypothesis: Machines that act intelligently necessarily:
  • are thinking
  • possess consciousness.


CI

A computational intelligence method is “an algorithm that learns to solve problems from observations (data) in a biologically inspired way”.


We get

  • inputs ﻿x={x1,x2,…,xn}x=\left\{x_1,x_2,\dots,x_n\right\}x={x1​,x2​,…,xn​}﻿ where ﻿x∈ℜnx\in\Re^nx∈ℜn﻿ (the input based on which we make the decision)
  • put them into a transfer function ﻿γ: ℜn→ℜm\gamma:\ \Re^n\to\Re^mγ: ℜn→ℜm﻿ with configuration parameters ﻿θ\thetaθ﻿ (the thing that needs to be learned)
  • which gives us outputs ﻿y={y1,y2,…,ym}y=\left\{y_1,y_2,\dots,y_m\right\}y={y1​,y2​,…,ym​}﻿ where ﻿y∈ℜmy\in\Re^my∈ℜm﻿ (the decision we have to make)


We make a distinction between a few types of learning problems:

  • Supervised learning: given inputs ﻿xxx﻿, predict output ﻿yyy﻿ based on given examples
  • Unsupervised learning: given inputs ﻿xxx﻿, predict output ﻿yyy﻿ by figuring out some hidden structure in ﻿xxx﻿
  • Iterative supervised learning (reinforcement learning): given inputs ﻿xxx﻿, predict output ﻿yyy﻿ (the action to take in state ﻿xxx﻿), then the correct ﻿yyy﻿ can be found, but is not known up front and will be approximated through exploration


A few parameter optimization methods are:

  • Gradient descent
  • Evolutionary approaches
  • Swarm intelligence
  • Ant colony optimizations
  • Analytic / brute force
