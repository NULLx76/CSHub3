Bidirectional associative networks

Bidirectional associative memory models try to simulate human memory; to access our memory we only care about the content, not the location. So the content triggers a memory which can be retrieved.


We model this the same as a feed-forward NN, but with a twist: we use the output we obtain as the input for the last layer, and we repeat this again and again until convergence:

As said before, we continue until we have converged, until there is an equilibrium. We perform the following computations:

  1. ﻿y0=sign(wT⋅x0)y^0=sign\left(w^T\cdot x_0\right)y0=sign(wT⋅x0​)﻿
  2. ﻿x1=sign(y0⋅w)x_1=sign\left(y^0\cdot w\right)x1​=sign(y0⋅w)﻿
  3. ﻿y2=sign(wT⋅x1)y_2=sign\left(w^T\cdot x_1\right)y2​=sign(wT⋅x1​)﻿

We find ﻿www﻿ by applying Hebbian learning: "Neurons that fire together wire together", so ﻿w′=w+x⋅yTw&#x27;=w+x\cdot y^Tw′=w+x⋅yT﻿. Here ﻿x⋅yTx\cdot y^Tx⋅yT﻿ is the correlation of the elements of the input and the elements of the output. Hopfield networks

Hopfield networks are a particular case of BA networks, where the amount of input and output neurons are the same. This is why they are called auto-associative memory networks. zThe goal of this is remembering inputs and getting the original input pattern from a corrupted input.


This means we can also represent this network as for example a triangle:

Image taken from here


Another characteristic is that the values are not updated synchronously, but asynchronously. In every iteration, we sample one neuron and update its values based on the other neurons.


The conditions for stability are:

  • It requires a zero diagonal; a neuron is not connected to itself so it doesn't influence its future values
  • It is symmetric; the matrix of weights needs to be symmetric


Without these requirements, the weights for the neurons would be cycling and it will never stabilize.


Learning

We learn by computing the following values: ﻿wij′=wij+xixjw_{ij}&#x27;=w_{ij}+x_ix_jwij′​=wij​+xi​xj​﻿ for ﻿i,j=1,…,ni,j=1,\dots,ni,j=1,…,n﻿ where ﻿i≠ji\ne ji​=j﻿.


In matrix form we write this as ﻿W′=W+x⋅xTW&#x27;=W+x\cdot x^TW′=W+x⋅xT﻿﻿−I-I−I﻿, to make sure that the diagonal remains zero.


We can compute the energy at a particular point by using the same energy function as for BA networks.


Self-organizing maps

Self-organizing maps will try to assign similar inputs to the same neurons.


To do that, they use competitive learning:

  1. Initialize weight vectors. Each of them is ﻿n×1n\times1n×1﻿ for ﻿nnn﻿ input elements
  2. Sample input vector
  3. Calculate distance between the weight vectors.
  4. Update weights: ﻿W′=W+α(x−W)W&#x27;=W+\alpha\left(x-W\right)W′=W+α(x−W)﻿


Maintain input topology

We want physically adjacent clusters to encode related information. When updating one neuron, we also update nearby neurons.
