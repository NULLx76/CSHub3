Regular expression & language equivalence

To prove that ﻿LLL﻿ is regular iff there exists a regular expression ﻿RRR﻿ such that ﻿L(R)=LL\left(R\right)=LL(R)=L﻿, we use 2 parts:

  1. For every regular expression ﻿RRR﻿, we construct an NFA ﻿MMM﻿ such that ﻿L(M)=L(R)L\left(M\right)=L\left(R\right)L(M)=L(R)﻿
  2. For every NFA ﻿MMM﻿ we construct a regular expression ﻿RRR﻿ such that ﻿L(R)=L(M)L\left(R\right)=L\left(M\right)L(R)=L(M)﻿


1.

Proof: We can prove this using induction on ﻿RRR﻿, by constructing an NFA that exactly one accept state

Base case: Construct ﻿M(∅)M\left(\varnothing\right)M(∅)﻿, ﻿M(ϵ)M\left(\epsilon\right)M(ϵ)﻿ and ﻿M(a)M\left(a\right)M(a)﻿ for each ﻿a∈Σa\in\Sigmaa∈Σ﻿

Inductive step: Given NFAs ﻿M(R1)M\left(R_1\right)M(R1​)﻿ and ﻿M(R2)M\left(R_2\right)M(R2​)﻿, construct ﻿M(R1∪R2), M(R1∘R2)M\left(R_1\cup R_2\right),\ M\left(R_1\circ R_2\right)M(R1​∪R2​), M(R1​∘R2​)﻿ and ﻿M(R∗)M\left(R^{\ast}\right)M(R∗)﻿


2.

To prove this, we use a generalized nondeterministic finite automata, in which the transitions are labeled with regular expressions over ﻿Σ\SigmaΣ﻿ instead of using symbols from ﻿Σϵ\Sigma_{\epsilon}Σϵ​﻿.


The finite automaton recognizing ﻿LLL﻿ is transformed into a GNFA in which subsequently all but the starting and accepting state are eliminated one by one. While doing this, the regular expressions on the transitions are modified such that the new GNFA recognizes the same language. Ultimately, the transition from the starting to the accepting state is labeled with the desired regular expression.


This GNFA is a special NFA where:

  1. the transitions are labeled with regular expressions over ﻿Σ\SigmaΣ﻿
  2. there exists exactly one start state of ﻿q0q_0q0​﻿
  3. there exist no transitions to the start state
  4. there exist no transitions from the accept state ﻿FFF﻿
  5. between every pair of nodes ﻿qi, qj∈Qq_i,\ q_j\in Qqi​, qj​∈Q﻿ there exists a transition such that ﻿δ(qi,qj)∈ℜ\delta\left(q_i,q_j\right)\in\Reδ(qi​,qj​)∈ℜ﻿ except for the cases mentioned above. Here, ﻿ℜ\Reℜ﻿ is the set of all regular expressions over ﻿Σ\SigmaΣ﻿


Definitions

A generalized transition graph is a transition graph belonging to a GNFA. In a generalized transition graph, edges labeled with ﻿∅\varnothing∅﻿ are usually left out. In the following, we do not discern a GNFA and its generalized transition graph.


A GNFA accepts a word ﻿www﻿ if in the generalized transition graph a path exists from the start state to the accepting state where the edges of the path are labeled with a sequence of regular expressions ﻿R1,…, RkR_1,\dots,\ R_kR1​,…, Rk​﻿, such that ﻿w∈L(R1∘⋯∘Rk)w\in L\left(R_1\circ\dots\circ R_k\right)w∈L(R1​∘⋯∘Rk​)﻿.


An NFA (transition graph) and a GNFA (generalized transition graph) are equivalent if they accept the same words.
