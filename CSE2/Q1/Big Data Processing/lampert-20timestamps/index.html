There is communication between 4 machines, A,B,CA, B, CA,B,C and DDD. Then the Lampert Timestamps would be as follows:

  • Initial state of timestamps: [A(0),B(0),C(0),D(0)][A(0), B(0), C(0), D(0)][A(0),B(0),C(0),D(0)]
  • AAA sends to CCC : [A(1),B(0),C(0),D(0)][A(1), B(0), C(0), D(0)][A(1),B(0),C(0),D(0)]
  • CCC receives from AAA: [A(1),B(0),C(2),D(0)][A(1), B(0), C(2), D(0)][A(1),B(0),C(2),D(0)]
  • CCC sends to AAA: [A(1),B(0),C(3),D(0)][A(1), B(0), C(3), D(0)][A(1),B(0),C(3),D(0)]
  • AAA receives from CCC: [A(4),B(0),C(3),D(0)][A(4), B(0), C(3), D(0)][A(4),B(0),C(3),D(0)]
  • BBB sends to DDD: [A(4),B(1),C(3),D(0)][A(4), B(1), C(3), D(0)][A(4),B(1),C(3),D(0)]
  • DDD receives from BBB: [A(4),B(1),C(3),D(2)][A(4), B(1), C(3), D(2)][A(4),B(1),C(3),D(2)]
