Sequences / lists

Processing big lists means splitting up those lists over many machines, which individually do processing over their part of the list. That's why in big data, immutable lists are used.


If it would be possible to change certain elements, everything would have to be synchronized again. When you can only append and prepend, only the first or last machine has to change.
