4.3 Synthesis of a negative feedback amplifier

If we want to build a negative feedback amplifier, without there being a circuit and simulations, we should do the following:

  1. Classify what we already have
  2. accurate devices, but they are constants so are bad for amplification
  3. inaccurate high gain devices, but they are inaccurate
  4. The only way to get an accurate, high gain device, we must use devices from both orthogonal categories


Let's start with accurate reduction, which can be done using a voltage divider. We will set a goal voltage (﻿UinU_{in}Uin​﻿) and add control to the voltage source (note this ﻿===﻿ is just a wish!):

To implement this, we will use nullators (equal sign) and norrators (a source that will provide whatever is necessary to make the nullator true)


Nullor

A nullor is a nulator and a norrator in a box. The nullator will have ﻿v=0, i=0v=0,\ i=0v=0, i=0﻿. The norrator will make sure this is always true, it will do whatever it takes to obtain these conditions.

So, a nullor is the simplest idealized model for an active circuit.

Using a nullor and passive components, you can build a perfect, accurate amplifier. Recall that active components can, generally speaking, inject power into a circuit and are capable of electrically controlling and amplifying the flow of electrical current, whereas passive components (such as a resistor) cannot.


Having a passive component in the active part (in the nullor area) has a price, since they only give a reduction, while the nullor gives a gain. So a passive component in the active part always makes the performance of the gain worse.


Operational amplifier

A nullor is an imaginary element that eases design, it is not an ideal operational amplifier:

Some differences:

  • Input resistance: undetermined (﻿00\frac{0}{0}00​﻿) for a nullor and infinite for an operational amplifier
  • Output resistance: undetermined for a nullor and zero for an operational amplifier


An ideal opamp would have the following characteristics:

  • Input resistance: infinite, but in practice it'll only be high
  • Output resistance: zero, but in practice it'll only be low


Design procedure

With this knowledge, we can describe an orthogonal design procedure:

  1. First use passive components, where we make an accurate reduction
  2. Then create dumb gain using a nullor (only using transistors or opamps)

We then end up with a gain of ﻿G=1B(AB1+AB)G=\frac{1}{B}\left(\frac{AB}{1+AB}\right)G=B1​(1+ABAB​)﻿, where the first term is the gain you'd have if this was an ideal case and the second term is the error.
