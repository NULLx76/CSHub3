Pulse Code Modulation (PCM)

  1. An analog value is represented by an ﻿nnn﻿-bits digital word, so we can have ﻿M=2nM=2^nM=2n﻿ discrete amplitude levels.
  2. We then sample this signal and turn it into time-discrete pulses
  3. The analog amplitude value is rounded to the nearest discrete value that can be represented (the closest value that we can represent with the ﻿nnn﻿ bits we have), quantizing. We use a flat-top PAM so we can get a stable output. The quantization error (yielding quantization noise) is at most half of a step size
  4. Since we have a finite number of discrete values, we can assign unique codewords to each amplitude: coding

PCM performance

With longer PCM words, the number of quantization levels increases, results in smaller errors, thus larger SNR.


For every extra bit:

  • The number of levels doubles
  • Maximum quantization error halves
  • ﻿10log⁡(4)10\log\left(4\right)10log(4)﻿﻿≈6 dB\approx6\ dB≈6 dB﻿ higher SNR

Advantages / disadvantages

Advantages

  • Cheap electronics
  • Transparent for different types of information: multiplexing of different types of data; analog as well as digital
  • Relatively insensitive for errors due to noise
  • Error detection/correction possible
  • Simple data encryption


Disadvantages

  • Requires much more bandwidth than needed for analog signals


Delta Modulation (DM)

In simple Delta Modulation, one bit per sample is transmitted indicating whether the current signal level is higher/lower than the reconstructed signal which is used as a reference level.


This reference level follows the input signals, so it is updated every signal it receives. Now you only have to transmit 1 bit: lower or higher than the input signal.



The reference starts at 0 and will be updated accordingly:


Noise sources

  • Slope overload noise / hunting noise: the reference signal can only change with step size ﻿δ.\delta.δ.﻿ If the input changes much faster, it will have trouble following the signal. Larger ﻿δ\deltaδ﻿ decreases this.
  • Granular noise: when the reference signal is only slightly different than the (slowly varying) input signal it will still update. Smaller ﻿δ\deltaδ﻿ decreases this.
