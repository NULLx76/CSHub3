Multilevel signaling

The amplitude of the orthogonal functions can take one of multiple discrete values (usually a power of 2). In this way multiple bits per symbol can be transmitted.


Multi-level signaling can be exploited to reduce the dimensionality of the signal and still transmit the same information:

  • in a shorter time than ﻿T0T_0T0​﻿ but using the same symbol rate and BW
  • in the same ﻿T0T_0T0​﻿ but using a lower symbol rate and less BW.

Transmission of digital signals in baseband

There are different line codes which represent binary 1's and 0's.


There are two main categories:

  • Non-Return-to-Zero (NRZ):
  • Return-to-Zero (RZ): the waveform returns to a zero-volt level for a portion of the bit interval


Desirable features of line codes:

  1. Spectrum adaptable to the available baseband channel
  2. Self-synchronization: timing information available also for long sequences of identical symbols
  3. Low bit error probability
  4. Small bandwidth: for a given bit rate
  5. Build-in error detection -correction
  6. Transparent and unambiguous

Differential coding

The data passed through circuits is sometimes unintentionally inverted. To make sure that this is not a problem, we can use differential coding: ﻿en=dn⊕en−1e_n=d_n\oplus e_{n-1}en​=dn​⊕en−1​﻿ (an XOR of the current input and previously transmitted channel symbol). This can be decoded as ﻿dn=en⊕en−1d_n=e_n\oplus e_{n-1}dn​=en​⊕en−1​﻿.


This means that the information is not coded int he symbol values, but in the transitions.

Inter-symbol interference

The absolute bandwidth of rectangular multilevel pulses is infinity. Removing part of the bandwidth causes the pulses to be rounded and causes them to cross over to the next symbol partially.


This distortion is called Inter-Symbol Interference (ISI).

For elimination of ISI, we use an equalizing filter: HR(f)=He(f)H(f)HT(f)HC(f)H_R(f)={H_e(f) \over H(f)H_T(f)H_C(f)}HR​(f)=H(f)HT​(f)HC​(f)He​(f)​.


Note that ﻿HT(f)H_T\left(f\right)HT​(f)﻿ is the transmitting filter, ﻿HC(f)H_C\left(f\right)HC​(f)﻿ are the channel characteristics (given), ﻿HR(f)H_R\left(f\right)HR​(f)﻿ is the receiving filter and ﻿He(f)H_e\left(f\right)He​(f)﻿ is the spectrum of the desired pulse shape (given).
