<p>You can view the web as a directed graph, where nodes are webpages and edges are hyperlinks.</p><p><br></p><p>People have tried to organize it by manually categorizing each website, but that was infeasible. Then came web search, which first tried to find relevant documents in a small and trusted set. But, the web is huge, so we have two challenges:</p><ul><li><strong>Who do we "trust"?</strong> Trick: trustworthy webpages may point to each other</li><li><strong>What is the "best" answer to a query? </strong>There is no single right answer. Trick: pages that know about this query might all be pointing to good results for this query</li></ul><p><br></p><p>Note that not all websites are equally important, so we need to rank the pages by the link structure (<strong>link analysis</strong>).</p><p><br></p><h1 id="basic-pagerank">Basic PageRank</h1><p>PageRank tries to determine whether a webpage is important.7</p><p><br></p><h2 id="flow-formulation">Flow formulation</h2><p>We think of links as votes: a page is more important if it has more <strong>in-links </strong>(links that point to this page). Approximately 10-15% of web pages are spam.</p><p><br></p><h3 id="term-spam"><strong>Term spam</strong></h3><p>Early search engines would build an inverted index of queries where for each query, an inverted index stores a list of relevant pages based on how often a query occurs in the page. This could be cheated by adding hidden words. This is <strong>term spam</strong>.</p><p><br></p><p>The solution to this was using the hyperlink text to find what a web page is about instead of believing the web page itself. Someone could still create false webpages which link to the page, but as these pages have no in-links their PageRank score will be low.</p><p><br></p><h3 id="spam-farm">Spam farm</h3><p>Spam farms were developed to concentrate PageRank on a single page, so a page which just contains hyperlinks.</p><p><br></p><p>From the point of view of a spammer, there are 3 types of pages:</p><ul><li>Inaccessible pages (pages a spammer cannot touch)</li><li>Accessible pages, e.g. blog comments</li><li>Own pages, completely controlled by the spammer</li></ul><p><br></p><p>If a spammer wants to maximize the PageRank of target page <span class="ql-formula" data-value="t">﻿<span contenteditable="false"><span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi>t</mi></mrow><annotation encoding="application/x-tex">t</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height: 0.61508em;
