<div class="markdown-body" style="white-space: normal;"><h1>Priority Queue</h1>
<p><em>This is about chapter 9.1 and 9.2 of the textbook</em></p>
<h2>Priority Queue Definition</h2>
<p>A prioerity queue is a collection of prioritized elements:</p>
<ul>
<li>Insert at any arbitrary position</li>
<li>Remove element with the first priority</li>
</ul>
<p>Priority queue entry: <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mo>(</mo><mi>k</mi><mi>e</mi><mi>y</mi><mo separator="true">,</mo><mi>v</mi><mi>a</mi><mi>l</mi><mi>u</mi><mi>e</mi><mo>)</mo></mrow><annotation encoding="application/x-tex">(key, value)</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:1em;vertical-align:-0.25em;"></span><span class="mopen">(</span><span class="mord mathdefault" style="margin-right:0.03148em;">k</span><span class="mord mathdefault">e</span><span class="mord mathdefault" style="margin-right:0.03588em;">y</span><span class="mpunct">,</span><span class="mspace" style="margin-right:0.16666666666666666em;"></span><span class="mord mathdefault" style="margin-right:0.03588em;">v</span><span class="mord mathdefault">a</span><span class="mord mathdefault" style="margin-right:0.01968em;">l</span><span class="mord mathdefault">u</span><span class="mord mathdefault">e</span><span class="mclose">)</span></span></span></span> where</p>
<ul>
<li>Key is the priority</li>
<li>Value is the element</li>
</ul>
<h3>In Java</h3>
<h4>Comparable</h4>
<p><strong>Natural ordering</strong>: a class defines a natural ordering of instances by implementing <em>java.lang.Comparable</em></p>
<h4>Comparator</h4>
<p><strong>External ordering</strong>: can be defined by another calss by implementing <em>java.util.Comparator</em></p>
<h2>Operations</h2>
<ul>
<li><em>insert(k,v)</em></li>
<li><em>min()</em>: returns and does not remove the element with the minimal key</li>
<li><em>removeMin()</em>: returns and removes the element with the minimal key</li>
<li><em>size()</em></li>
<li><em>isEmpty()</em></li>
</ul>
<p>If you use <em>min</em> and <em>removeMin</em> with elements that have the same key, the returned value is arbitrary</p>
<p><strong>Priority</strong>: A key associated with an element establishes its <strong>priority</strong>, usually numeric can be anything as long as there is a natural order.</p>
<p>For a comparison to be self-consistent, it must define a total order relation:</p>
<ul>
<li><em>Comparability</em>: The rule is defined for every pair of keys</li>
<li><em>Anti-symmetry</em>: If the rule holds independently of the order of keys, the keys must be identical</li>
<li><em>Transivity</em>: if <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi>a</mi><mi>R</mi><mi>b</mi></mrow><annotation encoding="application/x-tex">aRb</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.69444em;vertical-align:0em;"></span><span class="mord mathdefault">a</span><span class="mord mathdefault" style="margin-right:0.00773em;">R</span><span class="mord mathdefault">b</span></span></span></span> and <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi>b</mi><mi>R</mi><mi>c</mi></mrow><annotation encoding="application/x-tex">bRc</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.69444em;vertical-align:0em;"></span><span class="mord mathdefault">b</span><span class="mord mathdefault" style="margin-right:0.00773em;">R</span><span class="mord mathdefault">c</span></span></span></span> then <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi>a</mi><mi>R</mi><mi>c</mi></mrow><annotation encoding="application/x-tex">aRc</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.68333em;vertical-align:0em;"></span><span class="mord mathdefault">a</span><span class="mord mathdefault" style="margin-right:0.00773em;">R</span><span class="mord mathdefault">c</span></span></span></span></li>
<li><em>Reflexivity</em>: <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi>a</mi><mi>R</mi><mi>a</mi></mrow><annotation encoding="application/x-tex">aRa</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.68333em;vertical-align:0em;"></span><span class="mord mathdefault">a</span><span class="mord mathdefault" style="margin-right:0.00773em;">R</span><span class="mord mathdefault">a</span></span></span></span></li>
</ul>
<p>Minimal key is always: <span class="katex"><span class="katex-mathml"><math><semantics><mrow><msub><mi>k</mi><mrow><mi>m</mi><mi>i</mi><mi>n</mi></mrow></msub><mo>≤</mo><mi>k</mi></mrow><annotation encoding="application/x-tex">k_{min} \leq k</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.84444em;vertical-align:-0.15em;"></span><span class="mord"><span class="mord mathdefault" style="margin-right:0.03148em;">k</span><span class="msupsub"><span class="vlist-t vlist-t2"><span class="vlist-r"><span class="vlist" style="height:0.31166399999999994em;"><span style="top:-2.5500000000000003em;margin-left:-0.03148em;margin-right:0.05em;"><span class="pstrut" style="height:2.7em;"></span><span class="sizing reset-size6 size3 mtight"><span class="mord mtight"><span class="mord mathdefault mtight">m</span><span class="mord mathdefault mtight">i</span><span class="mord mathdefault mtight">n</span></span></span></span></span><span class="vlist-s">​</span></span><span class="vlist-r"><span class="vlist" style="height:0.15em;"><span></span></span></span></span></span></span><span class="mspace" style="margin-right:0.2777777777777778em;"></span><span class="mrel">≤</span><span class="mspace" style="margin-right:0.2777777777777778em;"></span></span><span class="base"><span class="strut" style="height:0.69444em;vertical-align:0em;"></span><span class="mord mathdefault" style="margin-right:0.03148em;">k</span></span></span></span> where <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi>k</mi></mrow><annotation encoding="application/x-tex">k</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:0.69444em;vertical-align:0em;"></span><span class="mord mathdefault" style="margin-right:0.03148em;">k</span></span></span></span> are all the other keys in the queue</p>
<h2>Implementations</h2>
<h3>Unsorted list:</h3>
<p><strong>Inertion</strong>: new entries  are added to the end of the list, in <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi mathvariant="script">O</mi><mo>(</mo><mn>1</mn><mo>)</mo></mrow><annotation encoding="application/x-tex">\mathcal{O}(1)</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:1em;vertical-align:-0.25em;"></span><span class="mord"><span class="mord mathcal" style="margin-right:0.02778em;">O</span></span><span class="mopen">(</span><span class="mord">1</span><span class="mclose">)</span></span></span></span> time</p>
<p><strong>Removal</strong>: Access (<em>min</em>) and removal (<em>removeMin</em>) of minimal key is done by traversing the list, <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi mathvariant="script">O</mi><mo>(</mo><mi>n</mi><mo>)</mo></mrow><annotation encoding="application/x-tex">\mathcal{O}(n)</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:1em;vertical-align:-0.25em;"></span><span class="mord"><span class="mord mathcal" style="margin-right:0.02778em;">O</span></span><span class="mopen">(</span><span class="mord mathdefault">n</span><span class="mclose">)</span></span></span></span> time</p>
<h3>Sorted list:</h3>
<p><strong>Insertion</strong>: entries kept in non-decreasing order of keys. Inertion needs traversal to find position, <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi mathvariant="script">O</mi><mo>(</mo><mi>n</mi><mo>)</mo></mrow><annotation encoding="application/x-tex">\mathcal{O}(n)</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:1em;vertical-align:-0.25em;"></span><span class="mord"><span class="mord mathcal" style="margin-right:0.02778em;">O</span></span><span class="mopen">(</span><span class="mord mathdefault">n</span><span class="mclose">)</span></span></span></span> time</p>
<p><strong>Removal</strong>: minimal key is always first int he list. Access (<em>min</em>) and removal (<em>removeMin</em>) are thus <span class="katex"><span class="katex-mathml"><math><semantics><mrow><mi mathvariant="script">O</mi><mo>(</mo><mn>1</mn><mo>)</mo></mrow><annotation encoding="application/x-tex">\mathcal{O}(1)</annotation></semantics></math></span><span class="katex-html" aria-hidden="true"><span class="base"><span class="strut" style="height:1em;vertical-align:-0.25em;"></span><span class="mord"><span class="mord mathcal" style="margin-right:0.02778em;">O</span></span><span class="mopen">(</span><span class="mord">1</span><span class="mclose">)</span></span></span></span></p>
