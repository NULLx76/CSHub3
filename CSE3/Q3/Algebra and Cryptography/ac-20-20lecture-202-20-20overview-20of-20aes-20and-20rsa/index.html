Block Ciphers

  • Block ciphers use symmetric ciphers, so the same key K is used for encryption and decryption
  • Block ciphers encrypt fixed-size blocks of plaintyext, and generates ciphertext blocks of the same length
  • Longer messages are encrypted one block at a time


Substitution-Permutation Networks (SPN)

  • One of the most common approaches to design block ciphers
  • The input block is encrypted in a confusion layer followed by a diffusion layer
  • The confusion layer is made of m Substitution boxes (S-boxes) ﻿Si : F2n  → F2nS_i\ :\ F_2^n\ \ \rightarrow\ F_2^nSi​ : F2n​  → F2n​﻿ applied in parallel, where mn = b and ﻿F2 = {0,1}F_2\ =\ \left\{0,1\right\}F2​ = {0,1}﻿ . Each S-box encrypts a sub-block of n bits
  • The diffusion layer is a Permutation Box (P-box) ﻿π: F2b →F2b \pi:\ F_2^b\ \rightarrow F_2^b\ π: F2b​ →F2b​ ﻿, which transposes the bit sin the whole block
  • The result is finally bitwise XORed with the round key ﻿kik_iki​﻿ (we assume I = b, thus ﻿P =C=K={0,1}bP\ =C=K=\left\{0,1\right\}^bP =C=K={0,1}b﻿ )

Concrete Block Ciphers

  • Block ciphers are the most commonly used symmetric cryptosystems in practice, since they are more versatile than stream ciphers
  • Typical block sizes and key lengths vary between 128 and 256 bits, to avoid brute-force attacks
  • Older block ciphers used smaller key lengths and are nowadays insecure, although they are based on a solid design
  • An example is the Data Encryption Standard (DES)



The Advanced Encryption Standard (AES)

  • In 1997 AES was the resultof a competition organised to select a new block cipher superseding DES
  • 15 candidate ciphers were submitted, out of which 5 finalists were selected:
  • Rijndael
  • Serpent
  • Twofish
  • RC6
  • MARS
  • In the end Rijndael was selected as the winner
